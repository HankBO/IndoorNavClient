# IndoorNavClient
MacOS client for dissertation 'Development of an Indoor Campus Navigation Solution based on Floorplan Transformation and Wi-Fi based Positioning'

Use CoreWLAN module in Swift to do network scanning

Backend Service Repository(develop based on [indrz-frontend](https://github.com/indrz/indrz-fe)): https://github.com/HankBO/indrz-frontend

Frontend Service Repository(develop based on [indrz-frontend](https://github.com/indrz/indrz-be)): https://github.com/HankBO/indrz-backend
